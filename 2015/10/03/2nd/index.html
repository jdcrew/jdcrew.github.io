我真的要更加努力/vim配置

这几天国庆都没有休息好，妹妹还有同学都来北京这边，其实我真的想休息休息的。
sad
今天出门之前找了点事做，因为平常基本都是用vim，所以把vim配置了一下，修改了一下vimrc。而且以后基本都要用vim来做C++的练习，当是为以后的练铺个路吧。
擦。。会话什么的看不懂啊！等我先学会其他的吧。。。
以后所有关于vim的博文都在这个标签下面更新了吧。

　　在vimrc中双引号开头的行都是注释
　　在vimrc中有以下可配置的属性，这些都参考自这里
　　　　1. set history =700 “vim的历史指令上限，注意等号和数字之间不要有空格，因为并不支持这样。
　　　　2. nmap <leader>w :w!<cr> “快速存储
　　　　3. set ruler “总是显示现在的光标位置
　　　　4. set hlsearch “高亮搜索结果
　　　　5. syntax enable “开启高亮功能
　　　　6. colorscheme desert “设置高亮主题为desert，也就是我现在用的主题
　　　　7. set shiftwidth =4 “设置一个tab为4个空格
　　　　8. set tabstop = 4
　　　　9. set ruler “这个是显示标尺
　　　　10. set foldenable “启动允许折叠
　　　　11. set foldmethod=manual “手动折叠
　　　　12. set helplang=cn “设置帮助语言为中文
