如何安装、简单配置以及完全卸载MySQL

MySQL的安装及简单配置

昨天接到傻逼的任务说要在我个人笔记本上面安装一个数据库，我当时的想法是拒绝的，是的，是拒绝的，因为觉得这个东西装到个人电脑总感觉卸载不干净啊，而且对数据库没有好感。

sad = = 好吧，那就开始吧。

1. 下载及安装MySQL

懒得找下载链接的可以在我这里下载,版本是5.6版本的，东西不大200+MB。

http://pan.baidu.com/s/1sjyzGw1

下载完之后打开安装文件
选择第一个开始

  • Install MySQL

然后

  • 同意条款

下一步的时候会选择要不要连接互联网,这个现在可以不用设置,等要设置的时候可以再去设置,所以两个都不要勾选,下面勾选check,让他不要自动检测更新。

  • 自动检测更新

然后我们就到了选择setup type这一步了，我看到大多的人都推荐custom，也有人认为默认的default会比较好一点，这里的选择决定了安装完成后的配置是怎么样的。我建议的话，其实也是我个人比较喜欢偷懒，喜欢一应俱全，所以选择了full这个type，如果你的计算机资源都比较足够的话可以选择full。这里要注意你自己的安装路径

  • Setup Type

然后一直Next到这一个位置:这里的Config Type是有三种选项的。

  1. Developer Machine: 也就是说你的机器只是用来作个人桌面工作站的。这个模式的配置会比较少的使用系统资源。
  2. Server Machine: 也就是说这是用于服务器的配置，可以同其他应用程序一起运行，如FTP、Email、Web Server。
  3. Dedicated MySQL Server Machine:说明是专用服务器，它为MySQL配置所有可用的系统资源。

我们选择第一种就可以了。

  • MySQL Server Configuration

接下来就是配置用户了，用户分为root用户和自定义用户组用户，建议的话add一个非root用户。
配置如图：

  • Add Users & Set Password

接下来就接近尾声了，命名你的Server Name
并且按照你个人意愿勾选是否开机启动。（很明显我个人很讨厌这个……sad）

  • Complete Your Server Configuration
  • Done

安装到此为止

3. 如何卸载MySQL

  1. 在程序和功能中执行uninstall程序(或者执行MySQL Installer 选取 remove MySQL)
  2. 删除\MySQL\MySQL Server 5.6\data文件夹(或者整个删除掉)
  3. 在regedit中删除注册表
